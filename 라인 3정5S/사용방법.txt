 라인 3정5S 관리 — 사용 / 배포 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

이 도구는 기존 "토크 측정 관리"와 똑같은 틀로 만들었습니다.
■ 클라우드(Firebase)는 토크 도구가 쓰던 프로젝트(cams-torque)를 그대로 재사용합니다.
   → 컬렉션(저장 공간)만 'fivees' 로 분리되어 있어 토크 데이터와 섞이지 않습니다.
   → 따라서 Firebase 추가 설치/설정이 전혀 필요 없습니다. 바로 폰↔PC 공유됩니다.


[ 1. 바로 써보기 (PC) ]
  · index.html 을 더블클릭하면 브라우저에서 열립니다.


[ 2. 폰에서도 보이게 하기 (GitHub Pages) ]
  토크 도구를 올린 것과 똑같이 GitHub 저장소에 올리면 됩니다.
  1) 이 폴더의 index.html 을 GitHub 저장소(예: Company-Profile-CAMS)에
     예: /라인3정5S/index.html  경로로 업로드
  2) 저장소 Settings → Pages 가 이미 켜져 있으므로
     주소: https://(아이디).github.io/(저장소)/라인3정5S/  로 접속
  3) 그 주소를 폰 홈화면에 추가하면 앱처럼 쓸 수 있습니다.
     (필요하면 현장용 QR 코드도 만들어 드립니다.)


[ 3. 사용법 ]
  · 메인  : 회사 레이아웃(구역 배치도) 등록, 월별·구역별 OK/NG 통계 차트
  · 점검 입력
      - 위 탭에서 [＋ 구역 추가] 로 구역을 만듭니다 (예: 조립 1라인, 부품창고, 검사장)
      - 구역을 고르고 [＋ 새 기록] 으로 점검 월을 만듭니다
      - 3정5S 항목(정품·정량·정위치 / 정리·정돈·청소·청결·습관화)을 OK / NG 로 판정
      - 조치 전 / 조치 후 사진을 촬영·첨부, 조치 내용 메모 입력
      - 모두 자동 저장 (저장 대기 → 저장됨 표시)
  · 이력 조회 : 월별·구역별 기록 보기, 수정, 삭제, 인쇄(PDF 저장)


[ 4. 관리자 비밀번호 ]
  · 레이아웃/고정사진 변경, 구역 이름변경·삭제, 기록 삭제 시 비밀번호 필요
  · 비밀번호: 103518  (토크 도구와 동일)


[ 주의 ]
  · 사진은 압축해서 클라우드에 저장합니다. 한 기록당 조치 전/후 각 6장,
    합쳐서 약 900KB 제한이 있습니다(꽉 차면 기존 사진을 지우고 추가).
  · 저장 주소를 아는 사람은 누구나 보고 수정할 수 있는 상태입니다(주소 비공개 전제).
    로그인/접근 제한이 필요하면 알려주세요.
